# Interactive Ivory Identification & Testing Showcase

An educational interactive exhibit that demonstrates the process for identifying and testing ivory objects through four distinct learning tracks.

## Overview

This interactive showcase allows visitors to choose how they want to engage with ivory object analysis, providing comprehensive information about material analysis, object typology, trade history, and ethics & law.

## How to Use

1. Open `index.html` in a web browser
2. Select one of the four tracks from the welcome screen
3. Navigate through the interactive content
4. Use the "Back to Tracks" button to explore other paths
5. Press ESC key to return to the welcome screen at any time
